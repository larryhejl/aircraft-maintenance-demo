User Guide

This guide covers everything you need to explore and analyze maintenance events in the dashboard.

Overview

  • Filterable event table — narrow down by aircraft, system, status, event type, or date range.
  • Event detail panel — full details for any selected event.
  • Charts — line chart of events over time and a status pie chart.
  • Unit Tests tab — run built-in tests and view results.

Getting Started

  1. Open the app in a modern browser (Chrome, Edge, Firefox, Safari).
  2. The dashboard loads 120 sample events spanning one year.
  3. Use tabs at the top to navigate between views.

Filters

FilterDescriptionOptions
AircraftAircraft typeAll, F-16C, F-15E, C-130J

Date RangeRecencyAll, Last 90 days, Last year
Tip: Filters combine — "F-16C" + "Engine" shows only F-16C engine events.

Events Table

Shows Date, Aircraft, System, Event Type, and Status. Click any row to see details below.

Charts

Events Over Time

Multi-line chart grouped by month. Use "Split by" to color by Aircraft, System, Event Type, or Status.

Status Breakdown

Donut chart showing Complete / In Work / Awaiting Parts proportions.

Unit Tests Tab

  1. Click "Run Unit Tests" to execute.
  2. View metrics, pass/fail chart, coverage ring, and detailed results table.
